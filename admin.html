오시는 길 · 네이버 지도

네이버 클라우드 플랫폼의 Maps API 키를 등록하면 실제 지도가 표시됩니다.

키 발급 — 네이버 클라우드 플랫폼 → Maps · Web Dynamic Map 등록 후 발급되는 Client ID(ncpKeyId)를 입력하세요. 콘솔의 Web 서비스 URL에 실제 배포 도메인을 등록해야 지도가 정상 표시됩니다.
